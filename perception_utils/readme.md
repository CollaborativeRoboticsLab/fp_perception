# perception_utils

`perception_utils` is a utility package for perception package. It manages 3rdparty packages and other helper functions and tools to support sensor data processing, object detection, and other perception modules.

## 3rdparty packages

### 1. facemesh 
originates from [KalanaRatnayake/Face_Mesh.Cpp](https://github.com/KalanaRatnayake/Face_Mesh.Cpp) which is a CPP replacement for Google Mediapipe face mesh
